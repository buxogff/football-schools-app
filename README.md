# საფეხბურთო სკოლების პორტალი

ვებაპლიკაცია საფეხბურთო სკოლების, გუნდების, სიახლეების, განრიგისა და შედეგების სამართავად.

## ტექნოლოგიები

- React + Vite
- Tailwind CSS
- lucide-react (აიქონები)
- localStorage — მონაცემები ინახება თქვენს ბრაუზერში (იხილეთ ქვემოთ მოცემული შენიშვნა)

## ლოკალურად გაშვება

```bash
npm install
npm run dev
```

შემდეგ გახსენით ბრაუზერში მისამართი, რომელსაც ტერმინალი დაბეჭდავს (ჩვეულებრივ `http://localhost:5173`).

## აწყობა გამოსაქვეყნებლად

```bash
npm run build
```

აწყობილი ფაილები აღმოჩნდება `dist` საქაღალდეში — ეს არის ის, რასაც ჰოსტინგ-სერვისი (მაგ. Vercel, Netlify) გამოაქვეყნებს.

## მონაცემების შენახვის შესახებ

ამჟამად აპლიკაცია მონაცემებს (სკოლები, გუნდები, მოთამაშეები, სიახლეები, განრიგი, შედეგები) ინახავს ბრაუზერის `localStorage`-ში. ეს ნიშნავს, რომ:

- მონაცემები რჩება მხოლოდ იმ მოწყობილობასა და ბრაუზერში, საიდანაც შეიყვანეთ;
- სხვადასხვა მოწყობილობიდან შესვლისას მონაცემები არ არის საერთო.

თუ მომავალში დაგჭირდებათ, რომ მონაცემები საერთო იყოს ყველა მოწყობილობისთვის (მაგალითად, რამდენიმე მწვრთნელი სხვადასხვა ტელეფონიდან ერთსა და იმავე მონაცემებს ხედავდეს), საჭირო გახდება რეალური სერვერული ბაზის დამატება (მაგ. Firebase ან Supabase).

## მობილურ აპლიკაციად დაყენება (PWA)

საიტს აქვს `manifest.json` და `service-worker.js`, რაც მას მობილურ ბრაუზერში ინსტალაციის საშუალებას აძლევს („დაამატე მთავარ ეკრანზე“). დეტალური ინსტრუქცია მოცემულია პროექტის განლაგების სახელმძღვანელოში.
